光融合产业

构建以MEMS(超小型精密机械技术)为基础的高附加值光半导体商用化基础设施
  • 项目期间 : 2020年~2022年
  • 总项目经费 : 100亿韩元
  • 项目主管 : 韩国生产技术研究院西南地区本部
  • 项目内容
    • 构建用于研发微型CWDM/传感器的必备MEMS工程设备
    • 研发湁MEMS基础的传感器/CWDM/光开关元件平台技术
    • 研发使用MEMS工艺的高附加价值元件，扶持试生产及人才培养
扶持光融合产业平台联合应用商业化
  • 项目期间 : 2019年~2021年
  • 总项目经费 : 160亿韩元
  • 项目主管 : 韩国光技术院，韩国电子通信研究院，韩国生产技术研究院
  • 项目内容
    • 扶持产品试验性生产/ 扶持样品生产， 满足光融合产业所需的产品功能和性能
    • 扶持光融合产品国际公认认证试验、测量〮分析等

  • 项目期间 : 2018年~2020年
  • 位置 : 光山区平洞产业园区模具特色园区内
  • 项目规模 : 占地面积3,302㎡，总面积2,640㎡

  • 项目内容

    • 运营模具制作革新设备及开发试验〮性能评估系统等
