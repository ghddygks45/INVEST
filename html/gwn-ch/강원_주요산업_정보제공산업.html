信息提供产业的特性与展望

何谓信息提供者(information provider：IP)
  • 指在 “原信息 → 信息提供者收集、 加工、 存储 (数据 化) 信息 → 信息通信手段 (电话、 PC 通信、互联网、文件等) → 信息需要者” 的连续过程中提供信息的企业或个人。
IP产业的特征
  • IP 产业收集和加工事业信息、就业信息、证券信息、不动产信息、专利信息、新闻、娱乐、游戏、各种生活信息并进行提供，这是一种极有前景的产业，只要构思新颖独特，任何人都可以进入
  • 随着宽带通信网的发展，信息通信服务竞争日益激烈，内容提供者 (contents provider ： CP) 或创造知识者成为开创附加价值的源泉。
  • 今后，内容的质量将决定收益性，因此 IP 将会向大型化发展， IP 虽然说进入费用低，但如果无条件创业，失败的可能性也很大。
国内信息提供产业现况及展望
  • 1990 年代后期， PC 通信企业的增加，出现了 netsgo、 Channel-i 等基于互联网的服务企业，目前，估计约有 2500-3500 家信息提供企业在提供 1500 多种信息服务。
    • 就韩国 IP 产业而言，其规模尚小，往往从属于大型 PC 通信企业。
  • 如果考虑到世界趋势， IP 产业今后在韩国也将从 PC 通信为主的信息提供向基于互联网的内容提供业 (CP) 扩张、转换。
    • CP 包括了可以利用互联网多媒体要素与超文本功能的多媒体内容服务，因此，与单纯提供文字信息的传统 IP 相比，是在技术上更加进步的形态。
  • 信息提供服务产业的市场规模从销售额来看， 1999 年增值通信服务与数据库制作合计达到 12959 亿韩元，与 1998 年的 8527 亿韩元相比，增长了 50% 以上。
  • 截至2000年1月，与信息提供服务有关的线路数量为53,367，自1997年以来，以每年平均约30％的速度快速增长。
    另外，截止 2000 年 1 月，信息提供服务网相关用户数达到 1860 万，如果以 1997 年的 850 万为基准，年均增长 46% 。
    • 其中，互联网接入服务用户显著增加， PC 通信用户增长相对较低，截止 2000 年 4 月，互联网用户超过了 1400 万。
  • 另一方面，截止 1998 年， DB 形态的信息提供产业销售额为 2158 亿韩元， DB 数为 3095 个， DB 制作机构数估计为 1357 家，整体而言，数据库市场保持年均 35% 的增长势头。
  • 按服务类别调整重复的 IP 后，截止 1999 年， IP 估计为 2850 家，销售额约为 6900 亿韩元，从业人员为 11500 人。预计到 2003 年， IP 数将年均增长 8%,销售额将增长 20%,从业人员增长 10% 左右。

江原地区的信息提供产业条件

信息提供产业现况及展望
  • 就江原地区而言，在企业、从业人员、产值等方面，信息提供事业基础非常薄弱。
    • 在增值通信服务企业数方面，截止 1998 年，全国为 3362 家，相反，江原地区只有 118 家 (3.3%),从业人数及产值方面分别为 1.8%、 0.6% 。
    • 在数据库制作方法，几乎不具备产业基础。
信息提供产业的发展条件
  • 与其它产业相比，信息提供产业创业更容易，成功可能性更大，从这点来说，具有可在任何地区发展的特性。
    • 创业费用平均仅为 2500 万韩元，进入及退出自由。
    • 即使没有技术或专利，只要有新颖的构思与机动性，一旦成功便可创造高附加值。
    • 随着服务基础从以往的 PC 通信向互联网转换，提供了广阔的事业机会。
  • 在信息提供产业基础薄弱的原州地区，信息提供产业虽然也可以发展，但成功的可能性不大。
    • 低进入费用意味着最终的激烈竞争，因此，为了成功，前提必须是构建差别化的竞争基础。
    • 不过，在竞争基础中最本质、最基本的是技术人才基础，就这点而言，原州地区可以说处于比较良好的状态。
  • 就原州地区而言， 3 所综合大学和 2 所专科大学培养出了电算相关人才，具有信息提供产业的发展潜力。
    • 原州地区的信息提供产业相关学生超过 1000 名，其中相当数量是由国立原州大学培养的。

战略事业的导出
  • 就信息提供业而言，尽管互联网及其它信息通信手段不断发展，但在信息的收集与加工、存储方面的事业基础薄弱的江原地区，是一个难以成功的行业。
    • 不仅信息的收集、加工、存储困难，即使可以实现上述工作，由于原信息的生产基础薄弱，难以获得信息。
    • 另外，信息提供业处于互联网 PC 通信企业等大型化趋势中，即使开拓夹缝市场，也难以与之竞争。
  • 但是，从广义上解释信息提供产业，如果部分执行信息提供功能，着重于信息内容制作，则存在某些程度的可能性。
    • 由于单纯的 IP 产业竞争激烈，属于失败率较高的产业，而且，附加值开创源泉摆脱单纯的信息提供，而是注重梯次内容生产。
    • 不过，在信息内容的制作方面，不同于单纯的 IP,需要相当数量的投资，因此需要有投资财源调配顺畅的前提。
  • 就原州地区而言，初期阶段最好注重符合地区特性的信息内容制作，而不是竞争激烈的高成长部门。
  • 如果原州市自身难以结成组合，由于首尔已经形成了专门投资医疗设备领域的 “无限医疗设备风险投资组合 Ⅱ”,可以积极利用上述组合。
    • 短期而言，最好以健康医疗、清净自然与农产品、丰富的天然草药等环保型清净特化作物相关信息为中心开始。
    • 长期而言，需要发展面向互联网 PC 通信事业者的多媒体内容开发，寻求与春川圈的多媒体内容产业衔接。
  • 而且，由于信息的 数据 化是信息提供产业的前提，应注重健康医疗、清净特化作物相关数据库
    • 构建数据库需要相当数量的技术与资本投入，因此，需要与江陵的软件振兴中心衔接，初期在公共部门创造需求。
    • 另外，信息提供产业受信息化影响较大，应持续推进计划中的地区信息化。
  • 在基础薄弱的原州圈，一开始就由企业直接向需要者提供信息比较困难，因此，初期最好采取向 (互联网)PC 通信事业者供应信息的形态。
  • 信息提供业在本质上具有风险企业的特性，因此，需要从培育风险企业的观点考虑产业发展。
    • 信息提供企业创业后直到通过技术开发实现新产品的商业化，需要相当长的时间，而且，这期间没有突出的收入源，会受到现金不足的困扰，因此，需要对该过程提供支援的体系。
    • 中央政府特别是信息通信部的各种产业支援计划不仅为个别中小企业，而且为研究所、团体等提供技术开发及事业化支援。
