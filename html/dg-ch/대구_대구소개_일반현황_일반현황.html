大邱的品牌形象

  • 色彩缤纷的大邱：意味着“多姿多彩”，有着年轻、明朗、漂亮、充满活力的城市形象，总括性、比喻性表现出多姿多彩、色彩缤纷的大邱未来面貌。
  • 蓝色 : 领导世界는 智慧经济城市
  • 绿色 : 健康、舒适的 环保绿色城市
  • 红色 : 洋溢年轻热情的 活力城市
  • 紫色 : 满怀创意和个性的文化艺术城市
  • 黄色 : 可患难与共的温暖的城市

一般现况

  • 位置 : 位于韩半岛东南部内陆的大邱市，是一个南北两面环山，东西是平缓的丘陵地带的盆地城市。与东南部海岸相隔80公里，与首尔相距293公里。
  • 面积 : 883.48㎢
  • 人口 : 2.518,000名(坐车一个小时车程内的人口为522万名，外国人数：25,203名 (1%))
  • 平均气温 : 14.0℃
  • 年均降水量 : 1,027.4 ㎜

产业现况

大邱市以良好的产业基础设施为背景，正在推进改编成尖端高附加价值型纤维服装产业的结构，并通过培养机械、金属与特化产业等地方传统主力产业的高附加价值与尖端知识产业，上调产业结构，正在向具备一定竞争力的城市方向发展。
