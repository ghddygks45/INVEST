済州特別自治道 未来戦略局未来戦略課スタートアップチーム

済州のスタートアップ政策を総括する済州特別自治道未来戦略局スタートアップチームはスタートアップのための規制革新と新産業育成の生態系を造成します。
  • 第4次産業革命ファンドの運営で新技術スタートアップの投資・育成
  • 第1次産業の育成に向けた農水畜スタートアップへの投資
  • 多様なインキュベーションプログラムの運営など
連絡先
  • TEL : +82 64 710-4811
  • 住所 : 済州特別自治道済州市新大路64 6階

済州創造経済革新センター

済州初のスタートアップ支援機関で、地域向けプログラムを通じた優秀スタートアップの発掘及び育成センターです。
  • 施設規模 : 済州ベンチャーマル 3、4階(1,924㎡)
    • 3階 交流及びネットワーク共同業務空間、在留者専用の業務空間、会議室、講義室、教育空間、ワンストップ法律・金融など相談空間
    • 4階 入居空間(45席)、Post BI卒業企業専用空間
  • 支援対象
    • 予備起業家又は起業3年以内のスタートアップ
  • 支援内容
    • The Edgeプログラム(6ヵ月)、Post BI空間支援(1年以内)、シードマネー投資事業の推薦、インキュベーション企業の基本支援及び特別支援、業務空間、共用空間など島外/海外スタートアップへの支援
    • 入居時に在留支援(1ヵ月範囲で最大5人)、済州本社登記移転費の支援
    • 住所: 済州特別自治道 済州市中央路217
    • ホームページ : http://jccei.kr

STARTUP BAY

西帰浦市のスタートアップを支援する起業支援機関で、西帰浦市のローカル産業と文化、観光、農業、海洋、ICTに特化された優秀スタートアップを発掘・育成しています。
  • 施設規模
    • 2階 コワーキングスペース(ネットワーキング及び教育)
    • 3階 入居事務室、セミナールーム、会議室
  • 入居支援
    • 入居企業20チーム、入居期間1年
  • 支援内容
    • 空間の提供(賃貸料、管理費無料)、教育及びメンタリング支援
  • 支援プログラム
    • 段階別のインキュベーション及びアクセラレーションプロセス
    • 国内外の優秀(予備)起業家の流入のための在留支援プログラム
    • デジタルノマドプログラム(50%起業教育、30%ローカルツアー、20%デジタルノマドデー)
    • 初期起業パッケージ(事業化資金 最大1億ウォン)
    • 起業生態系カンファレンスの開催、地域起業力量強化事業、初期創業企業の集中発掘及び育成
    • Tel : +82 64 732-0049
    • 住所:西帰浦市中正路86、3階

W360

古い気象庁ビルをリフォームした起業及び成長支援インフラの革新起業空間です。
  • 施設規模 : 959㎡
    • コワーキングスペース、講義室、会議室、入居空間(26席)、コンテンツ映像スタジオ、短期プロジェクト運営起業家空間など
  • 支援対象
    • 予備起業家又は起業7年未満の企業で、ビックデータ、ブロックチェーン技術を保有しているか当該技術を使用してサービスをしようとする個人及び法人事業者
  • 支援内容
    • 入居プログラム(2ヵ月)、シードマネー投資事業の推薦、インキュベーション企業の基本支援及び特別支援、業務及び共用空間の島外/海外スタートアップの支援
    • 入居時の在留支援(1ヵ月範囲で最大3人)、済州本社の登記移転費の支援
    • 住所:済州特別自治道 済州市東門路9ギル13-1
    • ホームページ : http://jccei.kr/wspace/intro.htm
  • W-SPACE : 誰でも自由に利用可能なコワーキングスペース
  • アルファα : 多様な教育及び行事が行われる講演場
  • デルタδ、エプシロンε : 予約を通じて自由に利用可能な会議室
  • ベターβ : 入居企業が固定的に利用可能な入居室
  • スタジオルーム : コンテンツクリエーターのための映像スタジオ
  • ガンマγ : 短期プロジェクトを運営する起業家のためのプロジェクトルーム
