外商投资企业

外商投资企业出于营利性目的而购置房地产的情况须遵守《关于房地产交易申报等的法律》、《外商投资促进法》、《房地产登记法》等。
  1. 外商投资企 业申报及登记
  2. 签订房地产 购置合同及付款
  3. 房地产 购置申报
  4. 房地产登记
  • 在外汇银行总行·分行或KOTRA进行外商投资申报及登记
  • 签订房地产购置合同后付款
  • 在房地产所在地的市、郡、区厅进行房地产购置申报。申报期限为自合同签订之日起60日内，提交材料为房地产购置合同。
  • 在土地所在地主管登记所办理房地产所有权转移登记，申报期限为自合同签署日或尾款支付日起60日内。所需材料包括法人登记簿副本（个人：外国人登录证复印件）、登记申请书、登记原因证明材料（盖章合同等）、登记权利证、房地产登记簿副本等。 代理人申报时，应补交委托书和代理人身份证。

外籍居民：外国人、外国法人的韩国分公司

外籍居民购置房地产时须遵守《房地产交易申报法》、《房地产登记法》等法律法规。
  1. 签订房地产 购置合同及付款
  2. 房地产 购置申报
  3. 房地产登记
  • 签订房地产购置合同并付款
  • 在房地产所在地的市、郡、区厅进行房地产购置申报。申报期限为自合同签订之日起60日内，提交材料为房地产购置合同。
  • 在土地所在地主管登记所办理房地产所有权转移登记，申报期限为自合同签署日或尾款支付日起60日内。所需材料包括法人登记簿副本（个人：外国人登录证复印件）、登记申请书、登记原因证明材料（盖章合同等）、登记权利证、房地产登记簿副本等。代理人申报时，应补交委托书和代理人身份证。

外籍非居民

外籍非居民购置房地产时须遵守《外汇交易法》、《房地产交易申报法》及《房地产登记法》等法律法规。
  1. 签订房地产 购置合同及付款
  2. 房地产 购置申报
  3. 房地产登记 申请注册号
  4. 房地产登记
  • 签订房地产购置合同
  • 支付房地产相关款项
  • 取出房地产购置资金时依照《外汇交易法》向外汇银行总行和分行进行房地产购置申报。所需材料包括房地产购置合同、房地产鉴定书或公示地价确认书、房地产登记簿副本。不仅申报房地产，还应申报房地产相关权利（物权、租赁权）。只有取得此申报书，以后房地产处置款项才能汇往海外。
  • 在房地产所在地的市、郡、区厅进行房地产购置申报。申报期限为自合同签订之日起60日内，所需材料为房地产购置合同。
  • 房地产登记专用登录号由居留地（在韩国没有居留地时，将居留地视为最高法院所在地）的地方出入境·外事机构负责人签发。所需材料包括：个人提交土地购置申报完毕证和护照复印件，法人=提交土地购置申报完毕证和所在国签发的法人登记证、代表人、代表人住址证明材料等。代理人申请时，应补交委托书和代理人身份证。在土地所在地主管登记所办理房地产登记，申报期限为合同签署日或尾款支付日起60天以内。所需材料包括分公司法人登记簿副本（个人：外国人登录证复印件）、登记申请书、登记原因证明材料（盖章合同等）、登记权利证、房地产登记簿副本等。代理人申报时，应补交委托书和代理人身份证。

永久居住权人

永久居住权人拥有韩国国籍，无论是否在韩国居住，均视同韩国公民。永久居住权人购置房地产时须遵守《房地产交易申报法》和《房地产登记法》等法律法规。
  1. 签订房地产 购置合同及付款
  2. 房地产 购置申报
  3. 房地产登记 申请注册号 (居民登记号码 清除后)
  4. 房地产登记
  • 签订房地产购置合同后付款
  • 在房地产所在地的市、郡、区厅进行房地产购置申报。申报期限为自合同签订之日起60日内，提交材料为房地产购置合同。
  • 如果身份证号码被注销，可通过首尔地方法院申请房地产登记专用登录号。所需材料为地址证明或居住事实证明、在外国民登录证。
  • 在土地所在地主管登记所办理房地产登记，申报期限为合同签署日或尾款支付日起60天以内。所需材料包括地址证明或居住事实证明、登记申请书、登记原因证明材料（盖章合同等）、登记权利证、房地产登记簿副本等。代理人申报时，应补交委托书和代理人身份证。
