就业

根据入境目的、所持签证类型，外国人也可在韩国就业。持韩国人配偶（F-13、F-21）签证入境的结婚移民者同样具有法定就业资格。请登录下列网站了解更多就业信息。
首尔国际中心 由首尔市运营的网站，面向在韩外国人提供博览会信息、就业支持、招聘信息等服务，服务语言包括英文及中文。
  • 网站 : http://global.seoul.go.kr
  • 电话号码 : 02-2075-4180
外国人雇佣管理系统 面向在韩业主和外籍劳动者提供招聘、就业信息的网站，主要服务有信访申请、申请状态查询等，同时介绍韩国雇佣许可制度、就业流程、可就业行业等信息，包括英文、中文、泰国语、越南语等20多个语种版本。面向普通外国人（E-9)、特例外国人（H-2)等不同签证种类，可提供不同服务。
  • 网站 : http://www.eps.go.kr
  • 电话号码 : 043-870-8000, 1577-7114
外国人综合服务中心 面向在韩外国人提供各种信息的门户网站，主要服务有在线信访、雇佣支持、投资支持等。
  • 网站 : http://www.hikorea.go.kr
  • 电话号码 : 1345

KOTRA雇佣推荐书制度

KOTRA实施的雇佣推荐书制度旨在向海外优秀技术人才聘用需求的公共/民营机构签发KOTRA社长名义的雇佣推荐书，帮助其获得特定活动签证（E-7)。
  • 电话号码 : 02-3460-7388, 02-3460-7387
  • E-mail : contactkorea@kotra.or.kr
高新技术人才资格条件
  • 具有相关领域硕士以上学历（不需要工作经验）
  • 具有相关领域韩国学士以上学历（不需要工作经验）
  • 具有相关领域学士以上学历+相关领域1年以上工作经验
  • 具有相关领域5 以上工作经验
※ 满足上述资格条件，并附上证明材料（学位或工作证明）的人才可申请雇佣推荐书。
可雇佣推荐书的行业 IT、技术经营、纳米、数字电子、生物技术、运输及机械、新材料、环境能源等8大高新技术领域
签发流程
  • 确认是否符合条件 : 确认雇佣推荐书对象行业及工种、以及各工种的被雇佣人开具条件。
  • 线上申请 : https://www.contactkorea.go.kr 注册企业会员，在线填写申请书，上传证明材料。
  • 材料审批 : 如要求补充材料时，需要修改申请书。
  • 查询履历 : 查询学历和经历。
  • 外部审查
  • 在官网下载雇佣推荐书
  • 提交到出入境事务所（申请E-7签证）
