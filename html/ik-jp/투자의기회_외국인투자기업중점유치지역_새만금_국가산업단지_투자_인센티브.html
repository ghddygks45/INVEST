セマングム開発事業はセマングム特別法、外国人投資促進法など多様な形態で投資家のためのインセンティブを提供しています。

外国企業向けの投資インセンティブ

北東アジア経済の新たな価値を創造するセマングム国家産業団地
外国企業向けの投資インセンティブ : 区分, 支援内容
区分 支援内容
国費(補助金) 立地支援
  • 資格 : 3,000万ドル以上の外国人投資
  • メリット : 無償賃貸
  • 支援程度 : 投資金額の50%以下
現金支援
  • 資格:外国人投資比率30%以上
  • 限度:外国人投資金額の5~40%(交渉による支援）
自治体(補助金) 一般
  • 立地 : 分譲価格の50%
  • 現金 : それぞれ最高50億ウォン(道·市)
大規模投資
  • 立地 : ）分譲価格の50%
  • 現金 : 1千億ウォン以上投資または300人以上雇用の際、最高400億ウォン支援
    ※ 現金支援から立地支援分を差し引く
  • 工場基盤 : 産業団地投資時、50億ウォンまで
  • 施設投資支援 : 敷地支援の場合、最高50億ウォン
雇用 20人を超える人数に限り、一人当たり150万ウォン*6ヶ月間、12億ウォン(道10、市2)
教育訓練 20人を超える人数に限り、一人当たり100万ウォン*6ヶ月間、7億ウォン(道5、市2)
国税(租税) 法人税 創業及び新設企業、5年間100%+2年間50%免除
(産業危機地域2022年4月4日まで)
関税 5年間100%減免(外国人投資として申告された輸入資本財)
地方税(租税) 取得税、財産税 15年間100%減免

インセンティブ事例

  • 東レ (2016)
    • 分野 : 自動車PPS樹脂及びコンパウンド
    • メリット : 100年間無償賃貸/法人税を5年間100%免除、2年間50%減免
  • ソルベイ (2016)
    • 分野 : エコタイヤシリカ
    • メリット : 現金支援/法人税を5年間100%免除、2年間50%減免
  • 電気自動車分野の強小企業8社(2019)
    • 分野 : バス、トラック、キャンピングカーなど
    • メリット : 補助金／法人税5年間100％免除
