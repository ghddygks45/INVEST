就労

外国人は、入国目的に応じて発給された査証の種類によって就労することができる。国民配偶者(F-13、F-21)査証を受けて入国した結婚移民者は法的に就労の資格が与えられる。下記のサイトを利用すればより簡単に就労関連情報が得られる。
ソウルグローバルセンター ソウル市が運営する就労関連サイト。韓国に在留中の外国人向けの就職フェア及び就労支援、採用情報を英語、中国語で提供している。
  • ウェブサイト : http://global.seoul.go.kr
  • 電話番号 : 02-2075-4180
外国人雇用管理システム 韓国居住の事業主と外国人労働者を対象に、各種の行政サービスの申請及び申請現況の照会など、採用/就労支援サービスを提供するサイト。雇用許可制、就労手続き、就労可能業種など全般的な情報を英語、中国語、タイ語、ベトナム語など20ヵ国語以上の言語で提供するホームページが設けられており、便利である。一般外国人(E-9)、特例外国人(H-2)など、査証の種類によって提供するサービスが異なる。
  • ウェブサイト : http://www.eps.go.kr
  • 電話番号 : 043-870-8000, 1577-7114
外国人総合案内センター 韓国に滞在する外国人のために様々な情報を提供するポータルサイト。電子行政サービス、雇用支援、投資支援など様々なサービスを提供する。
  • ウェブサイト : http://www.hikorea.go.kr
  • 電話番号 : 1345

KOTRA雇用推薦書制度

KOTRAでは、海外の優秀な技術人材を雇用しようとする公/私機関にKOTRA社長名義の雇用推薦書を発行し、特定活動査証(E-7)の発給を支援する雇用推薦書制度を実施している。
  • 電話番号 : 02-3460-7388, 02-3460-7387
  • E-mail : contactkorea@kotra.or.kr
先端技術人材の資格要件
  • 当該分野の修士号以上取得者(経歴不要)
  • 当該分野の韓国の学士号以上取得者(経歴不要)
  • 当該分野の海外の学士号取得者 + 当該分野の経歴1年以上保有者
  • 当該分野の経歴5年以上保有者
※ 本資格要件を満たし、証明書類(学位又は経歴証明書)を添付した人材に限り雇用推薦書の申請が可能
雇用推薦書申請会社の業種 IT、技術経営、ナノ、デジタル電子、バイオ、輸送及び機械、新素材、環境エネルギーの先端技術の8分野
発行手続き
  • 発行対象有無の確認 : 雇用推薦書の発行対象業種及び職種の確認、当該職種別の非雇用者の発行要件を確認
  • ホームページオンライン申請 : https://www.contactkorea.go.krに企業会員として加入、オンライン申請書作成、提出書類ファイルをアップロード
  • 書類検討 : 書類の補完を求められた場合、申請書を修正
  • 履歴確認照会 : 学歴及び経歴の照会
  • 外部審査
  • ホームページから雇用推薦書をダウンロード
  • 出入国事務所に提出 (E-7査証取得のための手続き)
