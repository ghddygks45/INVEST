第1条(目的)
此规则的目的是规定《关于忠清南道外国人招商引资的促进条例》的相关事项。<2008年7月30日 修订>
第2条 (定义)
此规则中使用的用语定义如下：
  1. 1. “入驻补助金”，是指从事制造业、高科技含量事业、产业支援服务业的总公司、研究所、工厂设施、旅游业等工作场所搬迁到道内外商投资区，或忠清南道知事(以下称“道知事”)为把外国资本引进道内而认为有必要且指定的地区，或在此新设、增设设施的情况，为以低价提供设施用地，资助部分出让价格，或希望获得比市价低的租赁费时，资助其差额。
  2. 2. 雇用补助金，是指总公司、研究所、工厂设施、旅游业等的场所搬迁到道内，或在道内新、增设投资，然后新雇用正式工作员工的情况下所支援的补助金。
  3. 3. 教育培训补助金，是指总公司、研究所、工厂设施、旅游业等的工作场所搬迁至道内，或在道内新、增设投资，然后为把新招聘的正式员工培养成符合企业经营活动的人才而实施教育培训时，用于支援其一部分费用的补助金。
  4. 4. 工厂设施，是指为在工厂用地内制造、加工或修理物品，而建立的必要建筑物（包括附带设施）、构造物与机械装置。
  5. 5. 长期正式员工，是指作为在相关工厂或工作场所以获得工资为目的而提供劳动的人员，根据《所得税法施行令》第185条第1项的规定，被记载在向管辖税务所提交的“所得税源泉征收履行状况申报书”上的最近3个月的平均劳动者数。
  6. 6. 总公司，是指在企业成立注册时，被明示的总公司或主要办公室所在的工作场所。
  7. 7. 所谓研究所，是指《技术开发促进法〉第7条所规定的企业附设研究所。
  8. 8. 投资补助金，是指向搬迁到地方的企业支援一部分建筑费、设施装备购买费、基础设施设置费的补助金。
第3条 (招商引资委员会的运营)
  1. ① 忠清南道招商引资委员会（以下称“委员会”）的委员长代表委员，统管委员会业务。
  2. ② 委员长召开会议时，必须在7天前书面通知各委员会议时间、场所与目的等，但是紧急情况下无须如此。
  3. ③ 委员长为运营委员会，必要时可邀请相关公务员或专家出席，听取意见或要求提供相关资料。
  4. ④ 在此规则中没有被规定的关于委员会运营的必要事项须通过委员会的审议，然后由委员长决定。
第4条 (招商引资实务委员会)
  1. ①关于忠清南道招商引资的促进条例》（以下称“条例”）的第3条第7项所规定的招商引资实务委员会（以下称“实务委员会”）包括委员长在内由20名以内的委员组成。<2008年7月30日 修订>
  2. ② 委员长为经济室长，委员是道知事从以下各项中选出后任命，或考虑到男女均衡，进行委任的。<2017年2月28日，2019年12月30日 修订>
    1. 1. 招商引资部门的所属公务员
    2. 2. 《外商投资促进法》附表1与附表2中所规定的关于处理信访事务的道、市、郡公务员
    3. 3. 具有丰富招商引资经验与见识的人员
  3. ③ 实务委员会由过半数委员出席而召开，出席委员过半数赞成而通过决议。
  4. ④ 委员长为运营实务委员会，必要时可邀请相关公务员或专家出席，听取意见或要求提供相关资料。
  5. ⑤ 为顺利运营实务委员会，安排一名干事，干事是招商引资负责官。
  6. ⑥ 实务委员会对以下事项进行审议：<2008年7月30日 修订>
    1. 1. 国外招商引资活动与支援<2013年8月30日 修订>
    2. 2. 审核、调整在委员会审议的案件
    3. 3. 处理国外投资者或投资企业的困难事项<2013年8月30日 修订>
    4. 4. 处理《外商投资促进法》第17条所规定的信访事务
    5. 5. 委员会委任的其它事项
第5条 (负责管理基金的公务员)
  1. ① 为有效管理、运营条例第8条规定的忠清南道投资振兴基金，设置基金运用官，基金运用官为经济室长。<2017年2月28日，2019年12月30日 修订>
  2. ② 使用、运作基金须遵循《忠清南道综合管理基金的设置与运用条例》。<2008年7月30日 修订>
第6条 (入驻补助金)
  1. ① 依据《条例》第11条第2项的规定，支援租赁费差额的期间从最初支付日起，不能超过10年。
  2. ② 根据《条例》第11条第3项的规定，所支援的土地出售价差额须按照《关于搞活产业集成与成立工厂的法律》第8条的规定，不能超过知识经济部长官所通告的“各行业基准工厂面积率”的工厂土地面积。>
  3. ③ 第2项所规定的补助金在地皮市价50%的范围内，可按照外资企业购买地皮后的投资实绩来进行事后支援，同时获得该项补助金支援的投资者，在无正当理由的情况下，必须经营10年以上申请补助金时所提交的“项目计划书”上的项目。
  4. ④ 希望获得补助金支援的投资者必须在签订租赁与土地购买协约日起1年内，向道知事提交附表1号表格的申请书。
第7条(现金资助的用途)
《条例》第10条的2所规定的现金支援只限用于《外商投资促进法施行令》第20条的2所规定的土地购买费或租赁费、雇用与教育培训补助金、建筑费、生产资料与研究器材购买费、电力与通信设施等基础设施的设置费等。
第8条 (雇用补助金)
  1. ① 条例》第12条所规定的雇用补助金是在外资企业注册后5年内，雇用在道内居住6个月以上的劳动者20名以上时，对超过的人员每人每月支援10万韩元至100万韩元的补助金。但是，雇用补助金由国家出资支援时，居住地区没有限制，根据上述标准，由委员会审议决定。
  2. ② 第1项所规定的补助金每个企业最多为3亿韩元，同时获得补助金支援的人员从获得支援日起3年内，必须维持原单位的雇用状态。
  3. ③ 希望获得补助金支援的人员必须在实施新员招聘的下一年度向道知事提交附表2号格式的申请书。<2004年7月20日 修订>
第9条 (教育培训补助金)
  1. ① 《条例》第13条所规定的教育培训补助金是在采用20名以上的国内人员后实施教育培训的情况下，6个月范围内，每人每月支援10万至100万韩元的补助金。
  2. ② 第1项所规定的补助金每个企业最多为3亿韩元，同时获得补助金支援的人员从获得支援日起3年内，必须维持原单位的雇用状态。
  3. ③ 第1项中所规定的教育培训补助金有以下几项：
    1. 1. 按照《劳动者职业能力开发法的施行令》第24条的规定，国家地方自治团体或公共团体设置、运营的公共职能开发训练设施（包括《技能大学法》规定的技能大学）。
    2. 2. 劳动部令所规定的职能开发训练设施与专业大学以上的教育机关，以及派遣到投资总公司的教育培训
    3. 3. 道知事认同的其它机关。
  4. ④ 第1项所规定的补助金支援范围等经委员会审议、决定。
  5. ⑤ 希望获得补助金支援的投资者必须在实施教育培训的下一年度，向道知事提交附表3号格式的申请书。
第10条 (设施补助金)
  1. ① 根据条例第14条，扶持的设施补助金的用途为以下各项中的任何一项。
    1. 1. 新设或增设工厂设施、研究设施或工作场所时所需要的建设费。
    2. 2. 以事业用火研究为目的而使用的资本材料、研究器材的购买费。
    3. 3. 电气、通信设施等基础设施设置费。
    4. 4. 此外，道知事认为有必要的设施。
  2. ② 设施补助金关于相关项目，从最初外商投资申报日起的5年内，新设或增设第1项中任何一项时，在必要费用总额的50%范围内可提供扶持。
  3. ③ 道知事如希望资助设施补助金时，对相关项目的投资支出计划的履行实绩进行评价，然后通过委员会的审议，提供扶持。
  4. ④ 希望获得补助金的人必须用附件第4号格式向道知事提交申请书。
  5. ⑤ 道知事从决定支付补助金日起，在预算范围内，可在10年以内分期支付补助金。
  6. ⑥ 道知事根据第5项，分期支付补助金时，考虑到投资计划变更或补助金执行实绩等，可调整金额或扶持时间。
第11条(扶持改善外国人的生活环境)
  1. ① 根据《条例》第15条的规定，为改善外国人的生活环境，可被支援的项目费有以下几种：<2008年7月30日 修订>
    1. 1. 新成立外国人专用学校时(包括扩建的情况)，在项目费与运营费总金额的50%范围内进行扶持，最高为2亿韩元
    2. 2. 建立外国人居住园区时，在土地购买的20%范围内，最多为2亿韩元
    3. 3. 为新建立服务支援项目而购买土地或新建大楼时，在项目费20%的范围内，最多为2亿韩元，或者在购买基本建筑物时，在购买费的20%范围内，最多1亿韩元。
  2. ② 希望获得第1项所规定的补助金的投资者在项目完工后1年内，向市长、郡守提交附表第7号格式的申请书，市长、郡守再将此申请书提交给道知事。
第12条 (投资企业的事后管理)
  1. ① 道知事为在事后管理获补助金支援的企业，可以对以下事项进行审核。
    1. 1. 项目计划的推进状况
    2. 2. 支援金额是否适当使用
    3. 3. 是否有取消、返还支援等事由
    4. 4. 为实现其它补助项目，判断有必要的事项
  2. ② 道知事根据第1项的规定，在经营实态检查结果中，如有被判定补助金使用于其它目的或补助金项目目的难以达成时，6个月范围内，可下达变更项目计划、要求施政、履行业务等指示。但是，如被判定有不得已的理由时，按照相关企业的申请，该履行期可在原要求的时间范围内延长一次（仅限一次）。
  3. ③ 道知事判断必要时，可要求市长、郡守提交获得支援企业的事后管理书，同时还可要求相关公务员到现场核实。
第13条 (补助金的计算等)
  1. ① 根据《条例》与《规则》的规定，获得补助金支援的投资者必须在附表第6号格式即关于相关年度补助金执行状况的核算书与支援决定书上明示项目进行状况，并在下一年度的1月末之前报道知事。
  2. ② 道知事在收到第1项的补助金核算书时，必须审查、核算补助金。
第14条 (取消与返还支援等)
  1. ① 道知事根据第6条的规定，在工厂开工后或公司（工厂）成立日起10年内，出售获得补助金后购买的用地时，必须按照开工或项目期间比率要求返还补助金。
  2. ② 道知事根据第7条、第8条的规定，3年内解聘获支援的员工时，必须按照雇用时间比率要求返还补助金的部分或全部。
  3. ③ 道知事根据第14条的规定，补助金核算结果显示项目费核算金额与原补助项目量相比有所减少时，必须要求返还支援补助金中相当于减少量的补助金。
  4. ④ 道知事在受到第13条第2项所规定的施政命令的投资者不执行命令时，必须要求返还使用于补助金目的外或相当于不正当使用部分的补助金，同时中断追加支援。
  5. ⑤ 道知事对于获得补助金支援的企业在签署完土地等出售、购买或出租合同后1年零6个月内没有动工建设的情况，可要求返回补助金。
第15条(支付成果奖金)
  1. ① 条例》第25条第1项所规定的成果奖金支付标准同附表1。
  2. ② 根据《条例》第25条第2项的规定，对刺激投资增长的公务员赋予人事上的优待与支付奖金，其标准同附表2。
  3. ③ 第1项至第2项所规定的有引资实绩的人员如希望获得奖金，必须在事情发生后6个月内，提交附表第8号格式的申请书
  4. ④ 是否支付奖金与金额数等须经委员会或忠清南道功绩审查委员会审议、决定。但是，人事上的优待则由委员会向相关机关、团体人事部建议。
第16条 (其它条例等的遵循)
关于执行补助金，在此规则中没有被规定的事项须遵循《忠清南道补助金管理条例》和同条《施行规则》。
