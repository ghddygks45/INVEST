大邱のブランドイメージ

  • 'Colorful DAEGU' : 「色彩に富んだ、多彩な」を意味し、若く、明るく、素敵で、華やかで、活気溢れる都市のイメージを与え、多様な魅力の大邱の未来ビジョンを包括的かつ隠喩的に表現
  • 青色 : 世界をリードするスマート経済都市
  • 草綠 : 健康で快適な環境にやさしいグリーン都市
  • あか : 若さと情熱に満ちた活気のある都市
  • 紫色 : ウィンドウと個性あふれる文化芸術都市
  • 黄 : 加えて生きていく暖かい福祉都市

一般現況

  • 位置 : 南北は山、東西はなだらかな丘陵地に囲まれた盆地型の都市で、東南部海岸から80Km、ソウルからは293Kmの韓国東南内陸部に位置している。
  • 面積 : 883.48㎢
  • 人口 : 2.518千人 (車で1時間以内に到達できる人口522万人、外国人人口 : 25,203人(1%))
  • 年平均気温 : 14.0℃
  • 年平均降水量 : 1,027.4 ㎜

産業現況

大邱市は整備された産業インフラを基に、先端付加価値型の繊維・ファッション産業への構造改変を図り、機械・金属及び特化産業の育成など大邱市の伝統的な注力産業の高付加価値化と先端知識産業の育成を通じて産業構造を高度化することで競争力のある都市に生まれ変わっています。
